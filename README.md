# Modul: AI-Augmented Product Delivery

## Projekt
Automatisches Cross-Selling-System für Industrie-Produkte.

## Funktion
- Liest Produkte aus SQL Datenbank
- Analysiert mit KI mögliche Ergänzungsprodukte aus ProduktKatalog 
- Schreibt Empfehlungen zurück in Datenbank
- Review & Improvement Loop
- Test Specification Generator (AI)
- Development Task Generator
- Playwright MCP

## Umsetzung  
- n8n (Workflow-Automatisierung)
- MySQL (Datenbank)
- OpenAI API (Produktanalyse)

## Ergebnis
Funktionierender Prototyp, der Produktdaten analysiert und Cross-Selling-Vorschläge generiert.
